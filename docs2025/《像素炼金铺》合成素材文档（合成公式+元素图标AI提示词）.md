# 《像素炼金铺》合成素材文档

## 一、物品合成公式（含合理性解释）

### （一）基础元素（4 种，初始解锁）

| 元素名称 | 解锁方式 | 用途                             |
| -------- | -------- | -------------------------------- |
| 水       | 初始解锁 | 合成液体类、降温类物品的核心元素 |
| 火       | 初始解锁 | 合成高温、能量类物品的核心元素   |
| 土       | 初始解锁 | 合成固体、矿物类物品的核心元素   |
| 气       | 初始解锁 | 合成气体、流动类物品的核心元素   |
